The DIVOC issued Verifiable Certificate has a digitally signed secure QR code. This can be authenticated online using the verification utility in this portal.

Steps for online verification:

  1. Visit https://verify.divoc.org/
  2. Click on “Scan QR” code
  3. A notification will prompt to activate your device’s camera. Point the camera to the QR code.

Please keep the following points in mind while scanning the QR code:

  • QR code should cover at-least 70%-80% of screen
  • Complete QR code should be part of camera frame
  • QR code should be parallel to the camera
  • Camera should be held steadily for at least 5 seconds
  • If camera is unable to read the QR code within 45 seconds, a message - “Camera is not able to read the QR code, please try again” with a try again button will be displayed. Verifier will be required to scan the QR code again following the instructions mentioned in Step 2.

On successful verification, the following will be displayed on the screen. (Refer to the image on the side)

  • Message “Certificate Successfully Verified”
  • Some of the attributes that will be displayed includes, but are not limited to:
    • Name
    • Certificate Issued On
    • Certificate ID
    • Valid From
    • Valid To
    • Issuer

In case of an unsuccessful verification, if the certificate is not genuine or is suspended, the screen will show the message Certificate Invalid or Certificate Suspended respectively. Refer to the images below.
